# Fidget Strings - Privacy Policy

Privacy policy page for the Fidget Strings iOS app.

## Live URL

After enabling GitHub Pages, your privacy policy will be available at:

```
https://[your-username].github.io/fidget-strings-privacy/
```

1. Create a new GitHub repository named `fidget-strings-privacy`
